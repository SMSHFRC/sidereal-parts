# 零件加工任務系統 — 成員使用手冊

> FRC 9501 · 網址 **https://part.team9501.org**
> 手機、平板、電腦都能用，建議在機台旁用手機操作。

---

## 1. 開始使用

### 登入
1. 手機瀏覽器開啟 **https://part.team9501.org**
2. 輸入**帳號**與**密碼**（向管理員索取；每人一組）
3. 進入後預設看到「任務看板」

> 第一次開啟若出現「後端啟動中」轉圈，是伺服器在喚醒（免費方案閒置會休眠），等約 30–60 秒即可，只有當天第一個人會遇到。

### 把網站加到手機主畫面（像 App 一樣）
- **iPhone（Safari）**：分享 → 加入主畫面
- **Android（Chrome）**：右上⋮ → 加到主畫面

之後點桌面圖示就能直接開，隊徽會顯示為 App 圖示。

---

## 2. 看懂任務看板（首頁）

任務依進度分成幾欄：

| 欄位 | 意思 |
|---|---|
| **待接受** | 還沒有人接的任務 → **這裡就是可以接的單** |
| **進行中** | 已被接走、正在加工或後處理 |
| **已完成 / 結案** | 做完、放棄或取消的 |

- **手機**：用上方的圓形按鈕切換欄位。
- 每張卡片顯示：零件編號（如 `ARM-0007`）、數量、加工方式、目前狀態。
- 標著 **🔓 開放接案** 的就是你可以接的單。
- 點任一張卡 → 進入**任務詳情**。

## 3. 接單與加工流程（核心）

### 情況 A：不需要後處理的任務

```
待接受 ──[你按 接受]──► 加工中 ──[你按 完成]──► 已完成（拿全部積分）
```

1. 在待接受欄找到想做的任務 → 點進去
2. 按 **接受** → 這張單就是你的了（別人看不到、也接不走）
3. 開始做 → 做的時候可以按 **開始加工**（狀態變「加工中」）
4. 做完 → 按 **完成** → 系統自動把積分加到你身上

### 情況 B：需要後處理的任務（例如陽極、噴砂）

後處理**可以是別人接手**。加工的人和後處理的人可以是不同人，積分也會分開算。

```
待接受 ─[接受]→ 加工中 ─[加工完成，交後處理]→ 後處理中 ─[完成]→ 已完成
        (你，加工者)          ↑拿加工積分          ↑後處理者拿後處理積分
```

1. 你接單、加工，做到「加工中」
2. 加工完成後，按 **加工完成，交後處理** → 你當場拿到**加工積分**，任務進入「後處理中」並回到公開池
3. 由**任何一位隊員**（可能是你、也可能是別人）到這張單按 **接下後處理**
4. 後處理做完 → 按 **完成** → 後處理的人拿到**後處理積分**

> 系統只會顯示你「現在這步能按的按鈕」，不用背流程——看到什麼按什麼就對了。

### 接了但做不了？
在「已接受 / 加工中」的任務按 **放棄回池**，任務會退回待接受，讓別人接。（放棄不扣分，但也不會拿到分。）

## 4. 積分怎麼算

完成任務就會加分，分數反映工作量：

```
任務總分 =（基礎 5 分 ＋ 有後處理再加 2 分）× 數量
```

| 例子 | 分數 |
|---|---|
| 做 1 個、免後處理 | 5 分 |
| 做 10 個、免後處理 | 50 分 |
| 做 4 個、要後處理 | 28 分 |

**拆帳規則**（有後處理時）：
- **加工的人**拿「基礎分 × 數量」（5/個）
- **後處理的人**拿「後處理分 × 數量」（2/個）

你目前的總分顯示在畫面**右上角**（例如「70 分」）。

## 5. 連結 Onshape（看零件 3D 縮圖）

任務若附了 Onshape 連結，連結你的 Onshape 帳號後，任務頁就會顯示零件縮圖，一眼看懂要做什麼。

1. 上方導覽列點 **Onshape** 按鈕
2. 跳到 Onshape → 用你的 Onshape 帳號登入 → 按 **Authorize（授權）**
3. 自動回到系統，按鈕變成 **已連結**
4. 之後打開 Onshape 任務就能看到縮圖，點「開啟 Onshape」可到原始 3D 檔

> 只需連結一次。你只看得到你 Onshape 帳號本來就有權限的文件——授權是唯讀的，系統不會改動你的 CAD。

---

## 6. 建立任務（如果你要出單給別人做）

1. 導覽列按 **新增**
2. 選：所屬系統、加工方式、數量（必填）；材料、後處理、繪圖連結、尺寸、備註（選填）
3. 繪圖連結可直接貼 **Onshape 文件網址**，系統會自動抓縮圖
4. 按 **建立任務** → 出現零件編號（如 `CHS-0003`）→ 任務進入公開池等人接

> 一般隊員建立的任務一律進公開池，由大家自行接單（只有管理員能預先指定給特定人）。

## 7. 常見問題

| 問題 | 解法 |
|---|---|
| 按「接受」出現「已被其他人接走」 | 有人比你快半秒，正常防搶單機制，換一張單 |
| 看不到某張任務 | 進行中的任務只有相關的人看得到；公開池的大家都看得到 |
| 忘記密碼 | 找管理員重設 |
| 第一次進站很慢 | 伺服器喚醒中，等約 1 分鐘 |
| 縮圖顯示不出來 | 先確認你已「連結 Onshape」；或你的帳號沒有該文件權限 |
| 手機按鈕太小 | 已針對手機優化，按鈕都夠大；若異常請回報管理員 |

## 快速小抄

| 我想… | 怎麼做 |
|---|---|
| 接一張單 | 看板「待接受」→ 點卡片 → **接受** |
| 回報加工完成（免後處理） | 任務頁 → **完成** |
| 交給別人做後處理 | 任務頁 → **加工完成，交後處理** |
| 接別人交出來的後處理 | 「後處理中」任務 → **接下後處理** → 做完 **完成** |
| 不做了、退回 | **放棄回池** |
| 看我的積分 | 右上角數字 |
| 看零件長怎樣 | 連結 Onshape 後，任務頁縮圖 |

有問題找管理員，或回報系統狀況。
